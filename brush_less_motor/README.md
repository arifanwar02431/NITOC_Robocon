# ブラシレスモータの制御
stand_alone.cpp: リポとESCを接続して回すだけ

with_controller.cpp: リポとESCに加え、SBDBT（usbドングル）とコントローラを使用

## 使用機器詳細
- ESC: Flycolor 40A OPTO brushless ESC 2-6S
- SBDBT: SBDBT 3v3 (usbドングル)
- コントローラー: PS3コントローラー (dualshock)
